# Software Engineering Training Roundup

The key takeaways

- Use a linter
- Atomic commits
- Write your code for maintainability - be kind to your future self
- Test critical parts of your code - ensure accuracy and reproducibility
- Use CI automation to enforce standards and to streamline processes
